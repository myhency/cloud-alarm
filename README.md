# Cloud-Alarm
알람관리 페이지

## 1. 프로젝트 구성
- React, Redux, Webpack, Babel, Jest, ESLint(Airbnb template), Material-UI

## 2. 테스트
```shell
npm install
npm test
```

```
npx jest --watchAll
```
- Unit Test 실행
- Unit Test 커버리지 점검(90% 이상)
- 인수 테스트 실행

## 3. 프로젝트 실행
```shell
npm start
```
- 기본 접속 URL: http://localhost:8080
